Element

class compas_assembly2.element.Element(name=None, id=None, frame=None, simplex=None, complex=None, insertion=None, **kwargs)[source]

Bases: compas.data.data.Data

Class representing a structural object of an assembly.

Parameters

  • name (str) -- e.g., "BLOCK", "BEAM", "FRAME" to be consistent majority of names are stored in compas_assembly2.ELEMENT_NAME class

  • id (list[int] or int) -- unique identifier/s , e.g.,0 or [0] or [0, 1] or [1, 5, 9]. one object can have an index and belong to a group/s

  • frame (Frame) -- local frame of the element there is also a global frame stored as an attribute

  • simplex (list[Polyline]) -- minimal geometrical represetation of an object type is a polyline since it can represent: a point, a line or a polyline

  • complex (list[any]) -- a list of geometries used for the element visualization currently supported types: geometry or Mesh

  • insertion (list[Vector, int]) -- direction of the element, often defined by a single vector (can be also a sequence) and an index in an insertion sequence

  • kwargs (dict, optional) -- additional keyword arguments can be passed to the element.

Attributes

  • frame_global (Frame) -- global plane that can be used for the element orientation in a larger assembly

  • aabb (list[Point]:) -- a list of XYZ coordinates defining the bounding box for collision detection.

  • oobb (list[Point]:) -- a list of XYZ coordinates defining the an oriented bounding box to the frame.

  • convex_hull (list[Mesh]:) -- a mesh computed from the complex geometries points

  • area (float) -- the surface are of an element based on complex geometry

  • volume (float) -- the volume of an element based on complex 3d geometry

Examples

>>> element = Element("BLOCK", 0, Frame.worldXY(), Point(0,0,0),                             Box(                                 frame=Frame([3, 0, 0], [0.866, 0.1, 0.0], [0.5, 0.866, 0.0]),                                 xsize=2, ysize=4, zsize=0.25),                             [Vector(0, 0, 1), 0])
